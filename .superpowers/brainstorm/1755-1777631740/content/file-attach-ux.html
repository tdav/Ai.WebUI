A — Скрепка в строке ввода

📎 открывает file picker. Прикреплённые файлы — chips над полем. Drag-and-drop на всю область чата. Компактно, не занимает место когда не нужно.

B — Компактная зона сверху

Полоска с drag-and-drop и кнопкой «выберите» постоянно видна, но занимает одну строку. Файлы отображаются прямо в ней.

C — Inline внутри поля ввода

Все файлы и скрепка внутри единого блока ввода — как в ChatGPT/Claude. Когда файлов нет, скрепка лежит рядом с полем. Максимально компактно.
